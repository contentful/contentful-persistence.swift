SyncSpacePersistable

public protocol SyncSpacePersistable: class

Your NSManagedObject subclass should conform to this SyncSpacePersistable to enable continuing a sync from a sync token on subsequent launches of your application rather than re-fetching all data during an initialSync. See Contentful’s Content Delivery API docs for more information.
